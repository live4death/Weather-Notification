# WeatherApp

WeatherApp is an application that shows the weather forecast for your city for the next 5 days.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install WeatherApp.

```bash
pip install -r requirements.txt
```
On Ubuntu/Debian, use venv or flag --break-system-packages

```bash
pip install -r requirements.txt --break-system-packages
```

## Usage

Run with python:

```python
python3 weather.py
```
or make it executable:

```python
chmode +x weather.py
```
and run as executable:

```python
./weather.py
```
